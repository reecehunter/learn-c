# Task
Write a program that defines, initializes, and prints variables for each of the following data types:
- char
- string (char array)
- int
- unsigned int
- float
- double
- long

# Example output
```
char c is X.
string str Hello world!.
int i is 10.
unsigned int ui is 5.
float f is 8.000000.
double d is 30.000000.
long l is 1000.
```
